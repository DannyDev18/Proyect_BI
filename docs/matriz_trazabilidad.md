## Reglas de Continuidad del Flujo

1. **Unidireccionalidad:** Nunca un "Endpoint de FastAPI" consultará el Modelo en crudo si el dato existe como una pre-asociación estática (Ej. K-Means escribe en `Dim_Cliente` mediante el ETL batch nocturno, con lo cual el API solo filtra contra SQL y no corre matemáticas costosas).
2. **Caché en Tiempo Real**: Componentes como _Cross-Selling_ y _Predicción de Metas_ utilizan matrices precargadas localizadas y en memoria RAM del Servidor (o un motor de inferencia efímero) dado el alto rendimiento y la variabilidad en los parámetros de entrada del usuario en el Frontend de React.
